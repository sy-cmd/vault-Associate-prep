# Vault Associate (003) — Exam Practical Requirements

## Linux Support

**Linux is NOT officially listed as a supported OS.** The exam runs on Certiverse, which officially only lists:

| OS | Supported Versions |
|----|--------------------|
| Windows | 10 / 11 or newer |
| macOS | Ventura or newer |
| Linux | **Not listed** |
| Chromebook | **Explicitly not supported** |

HashiCorp's own support docs say: *"most Linux users are able to take exams via Certiverse"* — but this is not a guarantee. Options:

1. **Email HashiCorp first** at certifications@hashicorp.com — describe your exact Linux setup and ask if it works
2. **Run a Windows VM** (VirtualBox/VMware) as a fallback — use Chrome inside it
3. **Run the Certiverse system compatibility check** in advance to catch problems before exam day

---

## Browser

- **Google Chrome** (latest) — recommended
- Microsoft Edge (latest) — also supported
- No special browser download needed — your existing Chrome is fine
- **Safe Exam Browser is NOT required** for HashiCorp exams
- **Disable pop-up blockers** before launching — they block screen sharing

---

## Hardware Requirements

| Requirement | Minimum |
|-------------|---------|
| CPU | Dual-core or better |
| RAM | 4 GB available |
| Disk | 1 GB free |
| Webcam | Required (internal or external, must be working) |
| Microphone | Required — **internal mic only, headset mics NOT allowed** |
| Download speed | 10 Mbps |
| Upload speed | 5 Mbps |

---

## ID

- One unexpired **government-issued photo ID** (passport, driver's licence, national ID)
- **No digital IDs** — must be physical
- Name on ID must **exactly match** the name in your HashiCorp certification portal account

---

## How to Register & Launch

1. Go to [cp.certmetrics.com/hashicorp](https://cp.certmetrics.com/hashicorp/) and log in with **GitHub**
2. Navigate to **Schedule/Take Exam**
3. Click **Certiverse SSO → Go to Certiverse Exam Platform**
4. Book your exam (cost: **$70 USD**)
5. On exam day, launch **15 min early** up to **45 min after** your scheduled time
6. If you abandon after launching, your voucher is **consumed** — no refund

---

## Other Rules

- **No VPN** — disable it before launching (some networks block Certiverse)
- **Quiet, private room** required — proctor can see your environment via webcam
- **Admin rights** needed on your machine
- **Keyboard**: US QWERTY layout recommended
