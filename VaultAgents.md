# VaultAgents
+ when ever we create an auth method with vault , the clients that gets the auth is the one that is responsible for keeping and using it in the lifecycle of token management

## vault agents
+ they provide a number of different helper features that address the following
 + auto authentication
 + storage for kepping tokens and delivary
 + lifecycle of tokens ( renewal and re-authentication )
+ The Vault Agent Auto-Auth functionality addresses the challenges related to obtaining and managing authentication tokens (secret zero).
