# AGENTS.md

This is a personal study repository for HashiCorp Vault Associate (003) certification. It contains:
- Study notes in Markdown files (`*.md`)
- Reference materials for Vault topics (auth, tokens, policies, secrets engines)
- A 2-week study plan and progress tracker

There is no code to build, test, or lint. OpenCode sessions should treat this as a reference/knowledge base, not a software project.
